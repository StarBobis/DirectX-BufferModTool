# DBMT

DBMT全新GUI界面，基于C# & WinUI3。

DBMT所有功能全部由用户设计，可以在issue中留下你的改进建议。

由于部分特殊原因，DBMT已转入闭源更新，只在Release中更新，不接受任何PR，有问题请提交到issue。
